# 图谱数据结构与接口

## 当前角色

本文件定义图谱接口概念层，适合决策、触发和进程安排读取关系与事件摘要。第一阶段真实存储落地时，以 `docs/12-人物与事件存储方案.md` 的 `Person`、`RelationshipEdge`、`RawEvent`、`SemanticEvent`、`FeedbackRecord` 为准。

本文件中的 `Event` 是“语义事件/图谱事件”的概念形态，不等同于原始聊天记录。原始聊天、网页片段、通知结果等应先进入 `RawEvent`，再提取或归并为 `SemanticEvent`。

## 接口边界

策略引擎只读取：

- 关系类型和行为准则摘要。
- 最近事件摘要。
- 健康度、阶段、趋势。
- 进程安排约束。

策略引擎不读取：

- 未脱敏原文。
- 无关隐私。
- 用户未授权导入的数据。
- P1 高风险事件的未确认推断。
