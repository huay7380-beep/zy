# 中国大陆运营合规与证件办理

本文件是执行前清单，不是法律、税务或报关意见。正式实施时，应由财税、海关、外汇、产品认证和目标国律师/代理复核。

## 主体路径

推荐使用中国大陆公司主体作为出口经营主体，便于开立对公账户、海关备案、出口退税/免税、收汇和与境外客户签约。

| 项目 | 目的 | 首期动作 | AI 节点 |
| --- | --- | --- | --- |
| 营业执照 | 形成合法经营主体 | 核对经营范围是否覆盖销售、进出口、互联网销售、供应链相关描述 | `cbx_01_entity_compliance` |
| 对公银行账户 | 收付款、外汇结算 | 开基本户/一般户，选择能办理贸易外汇名录的银行 | `cbx_11_contract_payment` |
| 税务身份 | 发票、申报、退免税 | 确认小规模/一般纳税人策略；外贸型/生产型出口退免税路径 | `cbx_13_customs_tax_fx` |
| 报关单位备案 | 进出口货物收发货人身份 | 通过多证合一、海关政务服务平台或中国国际贸易单一窗口办理 | `cbx_13_customs_tax_fx` |
| 电子口岸/单一窗口账号 | 报关、资质、查询、单证 | 准备法人/操作员账号、IC 卡或 USBKey、权限分配 | `cbx_13_customs_tax_fx` |
| 贸易外汇收支企业名录 | 货物贸易收付汇 | 首笔收支前在境内银行办理；小微跨境电商满足条件时可能免名录 | `cbx_11_contract_payment` |
| 出口退免税备案 | 退税、免税或免抵退 | 与税务顾问确认 0110、9710、9810、9610 对应路径 | `cbx_13_customs_tax_fx` |
| ICP 备案/许可 | 中国大陆服务器对外提供网站服务 | 若站点使用中国大陆服务器，需要备案；若仅海外服务器，按目标市场合规处理 | `cbx_04_independent_site` |
| 数据跨境合规 | CRM、广告、邮件、客服系统可能处理个人信息 | 识别是否涉及境内个人信息出境；建立最小必要和授权记录 | `cbx_15_audit_learning` |

## 外贸备案的变化

自 2022 年 12 月 30 日起，对外贸易经营者备案登记已取消。执行上不要再把“对外贸易经营者备案登记表”作为必办前置项；重点转为市场主体资格、海关报关单位备案、单一窗口/电子口岸、银行贸易外汇名录和税务备案。

## 海关监管模式选择

| 模式 | 典型含义 | 何时用 |
| --- | --- | --- |
| `0110` 一般贸易 | 传统出口报关 | 大货、线下合同、常规外贸订单 |
| `9610` 跨境电商零售出口 | B2C 小包直邮、清单核放、汇总申报 | DTC 零售、小额多频订单 |
| `9710` 跨境电商 B2B 直接出口 | 境内企业与境外企业达成交易后直接出口 | 独立站/平台 B2B 询盘成交，直接发客户 |
| `9810` 跨境电商出口海外仓 | 先出口到海外仓，再由海外仓交付买方 | 稳定 SKU、目的国库存、交付时效敏感 |
| `1210` 保税跨境贸易电商 | 特殊区域/保税场景 | 特定保税区业务，非首期默认 |
| `1039` 市场采购贸易 | 市场采购方式 | 小商品市场采购型贸易，视供货组织判断 |

首期如果是自有源头 + 境外 B2B 客户，优先设计 `9710/0110` 双路径；当目标市场复购稳定、库存周转可预测，再启用 `9810` 海外仓路径。

## 产品和目标国合规

每个 SKU 必须建立 `product_compliance_profile`：

| 维度 | 必填内容 |
| --- | --- |
| HS code 候选 | 中文品名、英文品名、材质、用途、图片、规格、归类依据 |
| 出口限制 | 是否涉证、涉检、两用物项、危险品、锂电、化学品、食品、医疗、动植物等 |
| 目标国准入 | EU CE/GPSR/REACH/RoHS、US FDA/FCC/CPSC/Prop65、UKCA、GCC、澳洲等 |
| 标签包装 | 原产地、进口商/责任人、警示语、批次号、说明书语言 |
| 测试报告 | 检测机构、报告编号、有效期、覆盖 SKU |
| 知识产权 | 商标、专利、版权、图片授权、平台侵权风险 |

## 收款与外汇

建议建立三层收款策略：

1. 小额样品：PayPal、信用卡、跨境收款账户或 TT 预付，注意手续费和拒付风险。
2. B2B 常规订单：TT、信用证、托收或平台担保交易；首单优先预付款。
3. 老客户账期：仅在信用评估、保险、历史付款和合同门禁通过后开放。

所有收款节点必须记录：

- 客户主体、付款人、发票/PI 编号、币种、金额、付款路径。
- 关联订单、报关单、物流单、收汇核销/外汇申报状态。
- 手续费、汇率、到账差额、退款或拒付风险。

## 税务策略

| 场景 | 税务关注 |
| --- | --- |
| 一般贸易/9710/9810 B2B | 通常按出口货物退免税通用规则设计，需报关单、进项票、收汇等证据 |
| 9610 零售出口 | 综试区内符合条件且无有效进货凭证的，可能适用增值税/消费税免税政策 |
| 综试区无票免税 + 企业所得税核定 | 需满足综试区注册、平台登记、电子商务出口申报等条件 |
| 海外仓 | 关注出口退税时点、海外库存、目的国 VAT/GST/销售税、进口商责任 |

AI 只能生成材料清单和风险提示，不能自动判定最终税务处理。

## 网站和数据

| 项目 | 要点 |
| --- | --- |
| 中国大陆服务器 | 通常需要 ICP 备案；经营性平台还可能涉及增值电信业务许可 |
| 海外服务器 | 面向海外客户可先用海外主机/CDN，但仍需处理支付、隐私、cookie、目标市场消费者法 |
| CRM/广告/邮件 | 若处理境内个人信息并向境外系统传输，要按中国数据跨境规则判断豁免、标准合同、认证或安全评估路径 |
| 客户授权 | RFQ、订阅、WhatsApp、邮件营销都要记录来源、同意、退订和隐私政策版本 |
