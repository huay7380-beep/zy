# 产品资料、图片和视频拍摄方案

跨境电商的产品内容不是美化材料，而是销售和合规证据。每个 SKU 应形成可复用的 `Product Content Kit`。

## 产品资料包

| 文件 | 用途 |
| --- | --- |
| 中文源资料 | 工厂内部规格、成本、材质、包装、工艺 |
| 英文规格书 | 给客户和站点使用 |
| 产品主数据 JSON | 给 AI、独立站、报价引擎使用 |
| 证书/报告 | 目标市场准入、客户信任 |
| 图片素材 | 主图、细节、场景、尺寸、包装、工厂 |
| 视频素材 | 15s/30s/60s、功能演示、装配、测试、包装 |
| FAQ | 客户常问：MOQ、样品、定制、交期、付款、物流 |
| 风险说明 | 易碎、危险品、锂电、尺寸偏差、颜色差异、售后边界 |

## 拍摄方式

### 1. 主图

目标：给独立站、Google Merchant、目录和平台主图使用。

- 白色或浅中性背景。
- 产品占画面主体，四周留安全边距。
- 不放促销字、水印、边框、夸张光效。
- 分辨率首期建议 2000 x 2000 以上，保留原片。
- 文件命名：`sku_angle_background_version.ext`，例如 `AB100_front_white_v01.jpg`。

### 2. 细节图

必须覆盖：

- 材质纹理。
- 接口/结构/功能点。
- 尺寸参照。
- 包装内容。
- 关键工艺。
- 缺陷边界或使用限制。

### 3. 场景图

用于广告、落地页、社媒和客户理解：

- 展示真实使用场景。
- 避免与产品无关的库存式背景。
- 不要让场景掩盖产品本身。
- B2B 产品优先展示应用行业、安装方式、生产/仓储/交付能力。

### 4. 证据图

给 B2B 信任背书：

- 工厂/产线/仓库。
- QC 检验。
- 包装流程。
- 样品打包。
- 认证报告局部脱敏展示。
- 装柜/发货。

### 5. 视频

建议每个核心 SKU 至少 3 类：

- `demo_15s`：广告短视频，展示核心卖点。
- `explainer_30s`：产品功能和适用场景。
- `factory_60s`：产能、质检、包装、发货能力。

## 拍摄器材最低配置

| 项目 | 建议 |
| --- | --- |
| 相机 | 手机旗舰机或微单 |
| 稳定 | 三脚架/俯拍支架 |
| 光源 | 两盏柔光灯或自然光 + 反光板 |
| 背景 | 白色背景纸、亚克力板、灰色布景 |
| 色彩 | 灰卡或固定白平衡 |
| 尺寸 | 尺子、手持参照、包装箱 |
| 后期 | 裁切、白平衡、去灰、压缩、文件命名 |

## AI 图片处理边界

允许：

- 去背景、裁切、提亮、压缩。
- 生成广告场景参考图。
- 生成文案、alt text、图片分类。

需要标注或谨慎：

- AI 生成/合成图用于广告或说明时，应保留元数据和内部标记。
- 不得把 AI 场景图当作真实使用证据。
- 不得修改产品结构、颜色、配件、认证标识，造成误导。

禁止：

- 编造证书、工厂、客户案例。
- 把未生产过的款式当作现货。
- 移除必须出现的警示标签或安全信息。

## SKU 素材验收

每个核心 SKU 首期至少：

- 1 张主图。
- 5 张细节图。
- 2 张场景图。
- 1 张尺寸图。
- 1 张包装图。
- 1 个 15-30 秒短视频。
- 1 份英文规格表。
- 1 份合规状态说明。

## 内容生成流程

```text
中文源资料
-> ProductMaster 草案
-> 拍摄清单
-> 原片入库
-> 图片质检
-> 英文标题/卖点/FAQ
-> 产品页草案
-> 合规和证据检查
-> 人工确认
-> 上线发布
-> 广告/询盘数据回写
```
